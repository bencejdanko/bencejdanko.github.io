Kong JWT Claim Injection Plugin

GitHub
This plugin enhances the standard Kong JWT authentication by automatically adding claims found within a validated JWT to the headers of the request sent to the upstream service.
  • Designed a modified Kong JWT Authentication Plugin Injecting serializable claims into upstream request headers securely.
  • Supported flat array layouts serializing JSON strings avoiding external Lua parsing library dependencies transparently.
